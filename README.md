# Dice Game 🎲

A simple dice game built using HTML, CSS and JavaScript.

## Features

- Two player game
- Roll dice
- Hold score
- Winning system

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Play

1. Click Roll Dice
2. Score increases
3. If dice shows 1, turn changes
4. Click Hold to save score
5. First player to reach 100 wins
